# 開源行為準則

在 Square，我們致力於對開源社群做出貢獻，並簡化發佈與管理開源軟體的流程。我們見證了來自數千名已經對我們專案做出貢獻的人們所展現出的巨大支持與熱情——我們希望確保我們的社群能持續對每個人真正開放。

本行為準則概述了我們對參與者的期望，以及檢舉不當行為的步驟。我們致力於為所有人提供一個溫馨且鼓舞人心的社群，並期望大家能遵守我們的行為準則。

Square 的開源社群力求：

 * **保持開放**：我們邀請任何人參與我們專案的任何方面。我們的社群是開放的，任何展現出所需能力與勝任力的貢獻者都可以承擔責任。

 * **周到考量**：人們會使用我們的成果，而我們也依賴他人的工作。在採取行動之前，請考慮使用者和同事。例如，對程式碼、基礎結構、政策和文件的變更可能會對他人產生負面影響。

 * **尊重他人**：我們期望大家共同努力解決衝突，假設出發點是良善的，並以同理心行事。請勿將分歧轉化為人身攻擊。

 * **通力協作**：協作可以減少冗餘並提高我們的工作品質。我們致力於在開源社群中保持透明，並與上游開發者以及自由軟體社群中的其他人密切合作，以協調我們的努力。

 * **務實求是**：我們鼓勵提問，且應在流程早期提問以避免日後出現問題。在尋找合適的論壇進行提問時，請深思熟慮。被提問者應給予回應並提供協助。

 * **體面地退出**：每個專案的成員都會有來有去。當某人離開或不再參與專案時，他們應該主動告知，並採取適當步驟確保他人可以接手他們留下的工作。

本準則並未詳盡無遺或完全涵蓋所有情況。其目的在於凝聚我們對協作、共享環境和目標的共同理解。我們期望大家在精神與字面上皆能嚴格遵守。

多樣性聲明
-------------------

我們鼓勵每個人參與，並致力於為所有人建立一個社群。雖然我們可能無法滿足每個人，但我們都同意每個人都是平等的。

每當參與者犯錯時，我們期望他們對此承擔責任。如果有人受到傷害或冒犯，我們有責任仔細且尊重地傾聽，並盡力糾正錯誤。

雖然此列表無法詳盡無遺，但我們明確尊重以下方面的多樣性：年齡、文化、族群、性別認同或表達、語言、國籍、政治信仰、職業、種族、宗教、性傾向、社會經濟地位和技術能力。我們不會容忍基於上述任何受保護特徵（包括身心障礙參與者）的歧視。

報告問題
----------------

如果您經歷或目睹了不當行為——或有任何其他疑慮——請透過發送電子郵件至 [codeofconduct@squareup.com][codeofconduct_at] 進行檢舉。如需更多細節，請參閱下方的報告指南。

報告指南
---------------

如果您經歷或目睹了不當行為——或有任何其他疑慮——請透過發送電子郵件至 [codeofconduct@squareup.com][codeofconduct_at] 進行檢舉。所有報告都將謹慎處理。

在您的報告中請包含：

 * 您的聯繫方式。
 * 涉事人員的名稱（真名、暱稱或化名）。如果有其他目擊者，也請一併提供。
 * 您對所發生事件的敘述，以及您是否認為該事件仍在持續。如果有公開紀錄（例如郵件清單封存檔或公開的 IRC 記錄器），請提供連結。
 * 任何可能有幫助的其他資訊。

提交報告後，Square 行為準則委員會的代表將親自與您聯繫。委員會隨後將審查該事件，追蹤任何額外問題，並決定如何回應。

任何被要求停止不當行為的人都應立即遵守。如果個人從事不當行為，Square 行為準則委員會可以採取他們認為適當的任何行動，最高包括在不經警告的情況下永久禁止參與所有 Square 空間。
