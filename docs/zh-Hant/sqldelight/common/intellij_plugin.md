# IntelliJ 外掛

IntelliJ 外掛為 `.sq` 檔案提供語言級別功能，包括：

 * 語法高亮
 * 重構/尋找使用
 * 程式碼自動補全
 * 編輯後產生 `Queries` 檔案
 * 右鍵複製為有效的 SQLite
 * IDE 中的編譯器錯誤可點擊跳轉至檔案
