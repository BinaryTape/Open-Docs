# IntelliJ 外掛程式

IntelliJ 外掛程式為 `.sq` 檔案提供語言層級的功能，包括：

 * 語法高亮顯示
 * 重構 / 尋找用法
 * 程式碼自動補全
 * 編輯後產生 `Queries` 檔案
 * 按右鍵即可複製為有效的 SQLite
 * IDE 中的編譯器錯誤可點擊跳轉至檔案
