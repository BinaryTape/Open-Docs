# 與 Coil 配合使用的項目

一系列能與 Coil 良好配合的第三方庫。

- [Telephoto](https://github.com/saket/telephoto)：用於在 Compose UI 中設計媒體體驗的建構組塊。
- [Landscapist](https://github.com/skydoves/landscapist)：一個可插拔式且高度優化的 Jetpack Compose 與 Kotlin Multiplatform 圖片載入程式庫，可透過 Glide、Coil 與 Fresco 獲取並顯示網路圖片。

您有與 Coil 配合使用或基於 Coil 構建的程式庫嗎？歡迎提交一個拉取請求 (PR) 將其收錄於此！
