Kotlin 基金會為 Google Summer of Code (GSoC) 提供導師指導，這是一項引導新貢獻者進入開源開發領域的全球線上計畫。

若要進一步了解 GSoC，請參閱 [Google Summer of Code 官方網站](https://summerofcode.withgoogle.com/)。

## GSoC 2026：專案構想
