[//]: # (title: Kotlin Night 指引)

Kotlin Night 是一種聚會 (meetup)，包含 3-4 場關於 Kotlin 或相關技術的演講。

> 有關 Kotlin Night 品牌資產，請參閱 [Kotlin 品牌資產文件](kotlin-brand-assets.md#kotlin-night-brand-assets)。
> 
{style="note"}

## 活動指引

* 請使用我們提供的 [品牌素材](kotlin-brand-assets.md#kotlin-night-brand-assets)。讓所有活動和素材保持相同風格，將有助於維持 Kotlin Night 體驗的一致性。
* Kotlin Night 應為免費活動。可收取少量費用以支應開銷，但應保持其非營利性質。
* 活動應公開發布，並開放所有人參加，不得有任何歧視。
* 如果您在活動結束後於網路上發布演講內容，必須是免費且所有人皆可存取的，無需任何簽署或註冊程序。
* 錄影是可選的，但我們建議進行錄影，並應將其公開。如果您決定錄製演講，我們建議制定計畫以確保品質良好。
* 演講應以 Kotlin 為主，不應專注於行銷或銷售。
* 活動可視情況提供食物和飲料。

## 活動需求

JetBrains 非常樂意支援您的 Kotlin Night 活動。因為我們希望所有活動都能提供同樣高品質的體驗，我們需要主辦單位確保滿足一些基本需求，以獲得 JetBrains 的支援。作為主辦單位，您需負責活動的以下方面：

1. 場地以及舉辦活動所需的一切，包括預訂舒適的場地。請確保：
    * 所有參加者都清楚活動的確切日期、地點和開始時間，以及活動時程與議程。
    * 有足夠的空間以及食物和飲料（如果您有提供的話）供每個人享用。
    * 您與講者已達成共識。這包括演講的時程、主題、摘要，以及簡報所需的任何設備。
2. 內容與講者
    * 歡迎邀請來自當地社群、鄰近國家甚至全球的講者。您的活動不一定要有 JetBrains 的代表或講者。然而，我們總是很樂意聽聞更多 Kotlin Night 的消息，因此歡迎隨時通知我們。
3. 公告與推廣
    * 請至少在聚會日期前三週發布活動公告。
    * 公告中應包含時程、主題、摘要和講者簡介。
    * 在社群媒體上廣泛宣傳。
4. 活動結束後向 JetBrains 提供活動素材
    * 我們很榮幸能在 [kotlinlang.org](https://kotlinlang.org/community/talks.html) 發布您的活動，如果您能提供投影片和影片素材以供後續發布，我們將不勝感激。

## JetBrains 支援

JetBrains 提供的支援包括：

* 授權使用 Kotlin Night 品牌（Branding），包括名稱與標誌（Logo）
* 周邊商品，例如給講者的貼紙與 T-shirt，以及給參加者的精美紀念品
* 在 Kotlin Talks 頁面列出該活動
* 如有需要，協助聯絡講者參與活動
* 盡可能協助尋找場地（透過聯絡人等），並協助媒合與當地企業的潛在合作夥伴關係
