[Kotlin Notebook](kotlin-notebook-overview.md) 是一個互動工具，讓您能夠在一份文件中混合程式碼、視覺效果和 Markdown。您可以利用 Notebook 來編寫並執行稱為「程式碼儲存格」的區塊中的程式碼，即時查看結果，並寫下您的想法。這種設定使其成為快速原型開發、分析和資料科學的絕佳工具。
