[Kotlin Notebook](kotlin-notebook-overview.md) 是一款互動式工具，可讓您在單一文件中混合程式碼、視覺化內容與 Markdown。
您可以使用 Notebook 在稱為「_程式碼資料格_」的區段中編寫並執行程式碼，即時查看結果並記錄您的想法。
這種設定使其成為快速原型開發、分析與資料科學的優質工具。
