[Kotlin Notebook](kotlin-notebook-overview.md) 是一款互動式工具，可讓您在單一文件中混合程式碼、視覺化內容和 Markdown。您可以使用筆記本在稱為 _程式碼儲存格 (code cells)_ 的區塊中編寫並執行程式碼、即時查看結果，並記錄您的想法。這種設置使其成為快速原型開發、分析和資料科學的極佳工具。
