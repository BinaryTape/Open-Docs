[//]: # (title: 型別總覽)

在 Kotlin 中，萬物皆物件，因為你可以在任何變數上呼叫成員函式和屬性。雖然某些型別在執行時會以最佳化的內部原始值表示 (例如數字、字元和布林值)，但對你而言，它們的表現和行為就像一般類別。

本節將說明 Kotlin 中使用的基本型別：

* [數值](numbers.md) 及其 [無符號對應型別](unsigned-integer-types.md)
* [布林值](booleans.md)
* [字元](characters.md)
* [字串](strings.md)
* [陣列](arrays.md)

若要了解其他 Kotlin 型別，例如 `Nothing`、`Any` 和 `Unit`，請查閱 Kotlin API 參考資料：

* [`Any`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-any/)
* [`Nothing`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-nothing.html)
* [`Unit`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-unit/)
