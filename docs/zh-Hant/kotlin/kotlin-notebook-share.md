若要分享 [Kotlin Notebook](kotlin-notebook-overview.md)，您可以直接將其上傳到任何 Notebook Web 檢視器，因為 [Kotlin notebook](data-analysis-overview.md#notebooks) 遵循通用的 Jupyter 格式。

我們推薦使用以下平台來分享 Kotlin notebook：

* **JetBrains Datalore：** 此平台不僅能方便分享 Kotlin notebook，還能提升其可用性。
  [Datalore](https://datalore.jetbrains.com/) 允許您執行與編輯 notebook，並整合了進階功能，例如建立互動式報告與排程執行 notebook。若要查看其實際運作，請參閱 [使用 DataFrame 的 Kotlin Datalore 範例](https://datalore.jetbrains.com/report/static/KQKedA4jDrKu63O53gEN0z/B5YeMMONSAR78FgKQ9yJyW)。
  ![Datalore Notebook 範例](datalore-example.png){width=700}
* **GitHub**：GitHub 原生支援渲染 Kotlin notebook，讓分享與協作變得簡單直接。
  範例請參閱 [Kotlin DataFrame GitHub 範例存儲庫](https://github.com/Kotlin/dataframe/blob/master/examples/notebooks/titanic/Titanic.ipynb)。
  ![GitHub Notebook 範例](github-notebook.png){width=700}
