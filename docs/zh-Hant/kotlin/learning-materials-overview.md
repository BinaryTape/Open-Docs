您可以利用以下資料與資源來學習 Kotlin：
* [基本語法](basic-syntax.md) – 快速了解 Kotlin 語法。
* [Kotlin 導覽](kotlin-tour-welcome.md) – 學習 Kotlin 程式語言的基礎知識。
* [慣用法](idioms.md) – 學習如何針對常見情境編寫慣用的 Kotlin 程式碼。
  * [Java 到 Kotlin 遷移指南：字串](java-to-kotlin-idioms-strings.md) – 學習如何在 Java 與 Kotlin 中處理常見的字串任務。
  * [Java 到 Kotlin 遷移指南：集合](java-to-kotlin-collections-guide.md) — 學習如何在 Java 與 Kotlin 中處理常見的集合任務。
  * [Java 到 Kotlin 遷移指南：可 null 性](java-to-kotlin-nullability-guide.md) — 學習如何在 Java 與 Kotlin 中處理可 null 性。
* [Kotlin Koans](koans.md) – 透過完成練習來學習 Kotlin 語法。每個練習都以失敗的單元測試形式呈現，而您的任務是使其通過測試。推薦給具備 Java 經驗的開發人員。
* [Kotlin Core 課程](https://hyperskill.org/tracks?category=4&utm_source=jbkotlin_hs&utm_medium=referral&utm_campaign=kotlinlang-docs&utm_content=button_1&utm_term=22.03.23) (由 JetBrains Academy 提供) – 在逐步建立可運作的應用程式時，學習所有 Kotlin 的要點。
* [Kotlin 書籍](books.md) – 查看我們已評選並推薦用於學習 Kotlin 的書籍。
* [Kotlin 技巧](kotlin-tips.md) – 觀看 Kotlin 團隊展示如何以更高效且慣用的方式使用 Kotlin 的短片，讓您在編寫程式碼時獲得更多樂趣。
* [Advent of Code 謎題](advent-of-code.md) – 透過完成簡短有趣的任務來學習慣用的 Kotlin 並測試您的語言技能。
* [Kotlin 動手實作教學](kotlin-hands-on.md) – 完成長篇教學以全面掌握一項技術。這些教學將引導您完成一個與特定主題相關的獨立專案。
* [Kotlin for Java Developers](https://www.coursera.org/learn/kotlin-for-java-developers) – 在 Coursera 上的這門課程中學習 Java 與 Kotlin 之間的相似與相異之處。
* [PDF 格式的 Kotlin 文件](kotlin-pdf.md) – 離線閱讀我們的文件。
