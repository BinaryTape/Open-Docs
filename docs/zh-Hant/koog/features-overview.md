# 總覽

代理功能提供了一種擴展和增強 AI 代理功能的方式。功能可以：

- 為代理添加新功能
- 攔截並修改代理行為
- 記錄和監控代理執行

Koog 框架實現了以下功能：

- [事件處理器](agent-event-handlers.md)
- [追蹤](tracing.md)
- [代理記憶體](agent-memory.md)
- [OpenTelemetry](opentelemetry-support.md)
- [代理持久化 (快照)](agent-persistence.md)
- Debugger
- Tokenizer
- SQL Persistence Providers
