# 總覽

Agent 功能提供了一種擴充與增強 AI Agent 功能的方式。功能可以：

- 為 Agent 增加新能力
- 攔截並修改 Agent 行為
- 記錄並監控 Agent 執行
- 在單一功能中為同一個事件類型註冊多個處理常式

Koog 架構實作了以下功能：

- [事件處理常式](agent-event-handlers.md)
- [執行緒](tracing.md)
- [Agent 記憶體](agent-memory.md)
- [OpenTelemetry](opentelemetry-support.md)
- [Agent 持久化 (快照)](agent-persistence.md)
- 偵錯工具
- Tokenizer
- SQL 持久化提供者
