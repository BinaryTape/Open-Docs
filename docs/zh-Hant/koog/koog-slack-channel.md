# Slack 上的 Koog

Koog Slack 頻道是 Kotlin Slack 工作區的一部分。

* 若您在該工作區已有帳號，請登入 [Kotlin Slack](http://kotlinlang.slack.com/) 並加入 `#koog-agentic-framework` 頻道。
* 若您還沒有帳號，請在 [瀏覽器](https://slack-chats.kotlinlang.org/c/koog-agentic-framework) 中開啟 Slack 頻道，或是在 [Kotlin Slack 註冊](https://surveys.jetbrains.com/s3/kotlin-slack-sign-up) 頁面提供您的電子郵件地址以申請邀請。
