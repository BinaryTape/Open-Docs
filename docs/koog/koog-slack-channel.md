# Slack 上的 Koog

Koog Slack 频道是 Kotlin Slack 工作区的一部分。

* 如果您已经在该工作区中拥有帐户，请登录 [Kotlin Slack](http://kotlinlang.slack.com/) 并加入 `#koog-agentic-framework` 频道。
* 如果您还没有帐户，请在[浏览器](https://slack-chats.kotlinlang.org/c/koog-agentic-framework)中打开该 Slack 频道，或通过在 [Kotlin Slack 注册](https://surveys.jetbrains.com/s3/kotlin-slack-sign-up)页面提供您的电子邮件地址来请求邀请。
