# 开源行为准则

在 Square，我们致力于为开源社区做出贡献，并简化发布和管理开源软件的过程。我们已经看到成千上万已经为我们项目做出贡献的人们所表现出的难以置信的支持和热情——我们希望确保我们的社区能够继续对每个人真正开放。

本行为准则概述了我们对参与者的期望，以及报告不可接受行为的步骤。我们致力于为所有人提供一个热情且鼓舞人心的社区，并期望我们的行为准则得到遵守。

Square 的开源社区努力做到：

 * **保持开放**：我们邀请任何人参与我们项目的任何方面。我们的社区是开放的，任何表现出所需能力和胜任力的贡献者都可以承担任何责任。

 * **体贴周到**：人们使用我们的成果，我们也依赖他人的成果。在采取行动之前，请考虑用户和同事。例如，对代码、基础架构、政策和文档的更改可能会对他人的产生负面影响。

 * **尊重他人**：我们期望人们共同努力解决冲突，设想良好的意图，并以同理心行事。不要将分歧转变为人身攻击。

 * **协同合作**：协作可以减少冗余并提高我们的工作质量。我们努力在开源社区内保持透明，并与上游开发者以及自由软件社区中的其他人紧密合作以协调我们的工作。

 * **讲求实际**：我们鼓励提问，并且应该在过程早期提问以避免以后出现问题。在为你的问题寻找合适的论坛时，请深思熟虑且体贴周到。被问者应积极回应并提供帮助。

 * **体面地退出**：每个项目的成员都会来来往往。当有人离开或脱离项目时，他们应该告知大家并采取适当步骤，以确保其他人可以接手他们留下的工作。

本准则并非详尽无遗或完备。它的作用是提炼我们对协作、共享环境和目标的共同理解。我们期望在精神和字面上都能遵守它。

多样性声明
-------------------

我们鼓励每个人参与，并致力于为所有人建立一个社区。虽然我们可能无法满足每个人，但我们都同意每个人都是平等的。

每当参与者犯错时，我们期望他们为此承担责任。如果有人受到伤害或冒犯，我们的责任是认真且尊重地倾听，并尽最大努力纠正错误。

尽管此列表无法穷尽所有，但我们明确尊重在年龄、文化、民族、性别认同或表达、语言、国籍、政治信仰、职业、种族、宗教、性取向、社会经济地位和技术能力方面的多样性。我们不会容忍基于上述任何受保护特征的歧视，包括对残障参与者的歧视。

报告问题
----------------

如果你经历或目睹了不可接受的行为——或有任何其他顾虑——请通过发送电子邮件至 [codeofconduct@squareup.com][codeofconduct_at] 进行报告。更多详情请参阅下方的报告指南。

报告指南
---------------

如果你经历或目睹了不可接受的行为——或有任何其他顾虑——请通过发送电子邮件至 [codeofconduct@squareup.com][codeofconduct_at] 进行报告。所有报告都将得到谨慎处理。

请在你的报告中包含：

 * 你的联系信息。
 * 相关个人的姓名（真实姓名、昵称或化名）。如果有其他证人，也请一并提供。
 * 你对所发生事情的叙述，以及你是否认为该事件仍在持续。如果有公开可用的记录（例如邮件列表存档或公开 IRC 日志记录器），请附上链接。
 * 任何可能有帮助的其他信息。

提交报告后，Square 行为准则委员会的代表将亲自与你联系。委员会随后将审查该事件，跟进任何其他问题，并决定如何回应。

任何被要求停止不可接受行为的人都应立即遵守。如果个人从事不可接受的行为，Square 行为准则委员会可能会采取他们认为适当的任何行动，包括但不限于在不经警告的情况下永久禁止进入所有 Square 空间。
