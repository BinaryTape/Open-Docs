# IntelliJ 插件

IntelliJ 插件为 `.sq` 文件提供了语言级功能，包括：

 * 语法高亮显示
 * 重构/查找用例
 * 代码自动补全
 * 在编辑后生成 `Queries` 文件
 * 右键点击以复制为有效的 SQLite
 * IDE 中的编译器错误可点击跳转至文件
