# Coil との連携

Coil とうまく連携するサードパーティライブラリのコレクションです。

- [Telephoto](https://github.com/saket/telephoto): Compose UI でメディア体験を設計するためのビルディングブロック。
- [Landscapist](https://github.com/skydoves/landscapist): Glide、Coil、Fresco を使用してネットワーク画像を取得・表示する、プラグイン可能で高度に最適化された Jetpack Compose および Kotlin Multiplatform 用の画像読み込みライブラリ。

Coil と連携する、あるいは Coil をベースに構築されたライブラリをお持ちですか？ここに追加するための PR を送信してください！
