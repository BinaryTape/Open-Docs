# 行動規範

## 私たちの誓約

オープンで歓迎される環境を育むために、私たちは貢献者およびメンテナーとして、年齢、身体の大きさ、障害、民族、性的特徴、ジェンダーアイデンティティと表現、経験レベル、教育、社会経済的ステータス、国籍、外見、人種、宗教、または性的指向に関わらず、すべての人にとって、私たちのプロジェクトやコミュニティへの参加をハラスメントのない体験にすることを誓います。

## 私たちの基準

ポジティブな環境づくりに貢献する行動の例には、以下のようなものがあります。

* 歓迎的でインクルーシブな言葉遣いをする
* 異なる視点や経験を尊重する
* 建設的な批判を寛容に受け入れる
* コミュニティにとって最善なことに焦点を当てる
* 他のコミュニティメンバーに対して共感を示す

参加者による容認できない行動の例には、以下のようなものがあります。

* 性的な言葉や画像の利用、および不適切な性的な関心やアプローチ
* トローリング、侮辱的・中傷的なコメント、および個人的または政治的な攻撃
* 公的または私的なハラスメント
* 住所や電子メールアドレスなどの他人の個人情報を、明示的な許可なく公開すること
* 専門的な場において不適切であると合理的に判断されうるその他の行為

## 私たちの責任

プロジェクトのメンテナーは、許容される行動の基準を明確にする責任があり、容認できない行動に対しては、適切かつ公正な是正措置を講じることが期待されます。

プロジェクトのメンテナーは、この行動規範に沿わないコメント、コミット、コード、Wikiの編集、Issue、その他の貢献を削除、編集、または拒否する権利と責任を有します。また、不適切、脅迫的、攻撃的、または有害であると判断した他の行動については、貢献者を一時的または恒久的に追放する権利と責任を有します。

## 適用範囲

この行動規範は、すべてのプロジェクトスペース内に適用されます。また、個人が公共の場でプロジェクトやコミュニティを代表している場合にも適用されます。プロジェクトやコミュニティを代表する例としては、公式のプロジェクト用メールアドレスの使用、公式のソーシャルメディアアカウントを介した投稿、またはオンライン・オフラインのイベントにおける任命された代表者としての活動などが挙げられます。プロジェクトの代表としての定義は、プロジェクトのメンテナーによってさらに詳細に規定され、明確にされる場合があります。

## 実施

悪用的、ハラスメント、またはその他の容認できない行動の事例は、プロジェクトチーム（colin at colinwhite.me）に連絡することで報告できます。すべての苦情は審査および調査され、状況に応じて必要かつ適切と判断される対応が取られます。プロジェクトチームは、事案の報告者に関する機密を保持する義務があります。具体的な実施方針の詳細については、別途掲示される場合があります。

行動規範を誠実に遵守、または実施しないプロジェクトメンテナーは、プロジェクトリーダーシップの他のメンバーによって決定される、一時的または恒久的な影響を受ける可能性があります。
