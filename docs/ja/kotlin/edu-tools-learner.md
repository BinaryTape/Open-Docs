[//]: # (title: JetBrains Academy プラグインで Kotlin を学習する)

[Android Studio](https://developer.android.com/studio) と [IntelliJ IDEA](https://www.jetbrains.com/idea/) の両方で利用可能な [JetBrains Academy プラグイン](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy)を使用すると、実践的なコード演習タスクを通じて Kotlin を学習できます。

[学習者スタートガイド](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy/docs/learner-start-guide.html?section=Kotlin%20Koans)をご覧ください。IntelliJ IDEA 内で Kotlin Koans コースを開始する手順を確認できます。
インタラクティブなコーディング課題を解き、IDE 内で即座にフィードバックを受け取ることができます。 

教育目的で JetBrains Academy プラグインを使用したい場合は、[JetBrains Academy プラグインで Kotlin を教える](edu-tools-educator.md) をお読みください。
