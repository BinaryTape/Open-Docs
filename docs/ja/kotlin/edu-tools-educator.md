[//]: # (title: JetBrains Academy プラグインを使用して Kotlin を教える)

[Android Studio](https://developer.android.com/studio) と [IntelliJ IDEA](https://www.jetbrains.com/idea/) の両方で利用可能な [JetBrains Academy プラグイン](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy) を使用すると、コード演習タスクを通じて Kotlin を教えることができます。

プログラミングタスクと統合テストのセットを含むシンプルな Kotlin コースを作成する方法については、[Educator Start Guide](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy/docs/educator-start-guide.html?section=Kotlin) を参照してください。

JetBrains Academy プラグインを使用して Kotlin を学習したい場合は、[JetBrains Academy プラグインで Kotlin を学習する](edu-tools-learner.md) をお読みください。
