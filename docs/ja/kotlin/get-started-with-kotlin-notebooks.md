[Kotlin Notebook](kotlin-notebook-overview.md) は、コード、ビジュアル、Markdown を 1 つのドキュメントに混在させることができるインタラクティブなツールです。 
ノートブックを使用すると、_コードセル (code cells)_ と呼ばれるセクションでコードを記述・実行し、その結果を即座に確認しながら、自分の考えを書き留めることができます。 
この構成により、プロトタイピング、アナリティクス、データサイエンスに最適なツールとなっています。
