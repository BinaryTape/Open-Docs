# 概要

エージェント機能は、AIエージェントの機能を拡張および強化する方法を提供します。機能により、以下のことが可能になります：

- エージェントへの新しい機能の追加
- エージェントの動作のインターセプトと変更
- エージェント実行のログ記録とモニタリング
- 単一の機能内で、同じイベントタイプに対して複数のハンドラーを登録

Koogフレームワークは、標準で提供されている機能の実装に加え、独自のカスタム機能を実装することも可能です。すぐに利用可能な機能には、以下のものが含まれます：

- [イベントハンドラー](agent-event-handlers.md)
- [トレーシング](tracing.md)
- [チャットメモリ](chat-memory.md)
- [エージェントメモリ](agent-memory.md)
- [OpenTelemetry](opentelemetry-support.md)
- [エージェントの永続化（スナップショット）](agent-persistence.md)
- デバッガー
- トークナイザー
- SQL永続化プロバイダー

独自の機能を実装する方法については、[カスタム機能](custom-features.md)を参照してください。
