# サンプル

Koogフレームワークは、さまざまなユースケースにおけるAIエージェントの実装方法を理解するのに役立つサンプルを提供しています。
サンプルは**Kotlin**と**Java**の両方で利用可能です。
これらは、独自のアプリケーションに適応できる主要な機能やパターンを示しています。

以下のサンプルを参照し、リンクをクリックしてGitHub上のソースコードを確認してください。
