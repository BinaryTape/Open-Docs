# オープンソース行動規範

Squareでは、オープンソースコミュニティへの貢献と、オープンソースソフトウェアの公開および管理プロセスの簡素化に尽力しています。私たちは、すでに私たちのプロジェクトに貢献してくださっている数千人もの方々から、多大なるサポートと熱意をいただいてきました。私たちは、私たちのコミュニティがすべての人にとって真に開かれた場であり続けることを願っています。

この行動規範は、参加者に期待される事項と、容認できない行為を報告するための手順を概説するものです。私たちは、すべての人にとって歓迎され、刺激となるコミュニティを提供することに尽力しており、この行動規範が遵守されることを期待しています。

Squareのオープンソースコミュニティは、以下を目指しています。

 * **オープンであること (Be open)**: 私たちは、プロジェクトのあらゆる側面への参加を歓迎します。私たちのコミュニティはオープンであり、必要な能力と適性を示した貢献者であれば、誰でも責任を担うことができます。

 * **思いやりを持つこと (Be considerate)**: 私たちは他者の成果を利用し、他者も私たちの成果を利用します。行動を起こす前に、ユーザーや同僚のことを考えてください。例えば、コード、インフラ、ポリシー、ドキュメントの変更は、他者に悪影響を及ぼす可能性があります。

 * **敬意を払うこと (Be respectful)**: 衝突を解決するために協力し、善意を前提とし、共感を持って行動することを期待します。意見の相違を個人的な攻撃に発展させないでください。

 * **協力的であること (Be collaborative)**: 協力は重複を減らし、成果の質を向上させます。私たちはオープンソースコミュニティ内での透明性を追求し、上流のデベロッパーやフリーソフトウェアコミュニティの他の方々と密接に連携して活動を調整します。

 * **実利的であること (Be pragmatic)**: 質問を奨励します。後々の問題を避けるため、プロセスの早い段階で質問してください。質問に適切な場を探す際は、思慮深く、配慮を持ってください。質問を受けた側は、迅速かつ有益に対応してください。

 * **配慮を持って身を引くこと (Step down considerately)**: すべてのプロジェクトにおいて、メンバーの入れ替わりはあります。プロジェクトを離れる、あるいは関与を止める場合は、その旨を周知し、他の人が後を引き継げるよう適切な手順を踏んでください。

この規範は、すべてを網羅しているわけでも、完全なものでもありません。これは、協力的な共有環境と目標に対する私たちの共通認識を抽出したものです。私たちは、文面通りであることはもちろん、その精神が守られることを期待しています。

多様性に関する声明
-------------------

私たちはあらゆる人の参加を奨励し、すべての人にとってのコミュニティを構築することに尽力しています。すべての人を満足させることはできないかもしれませんが、全員が平等であることには同意しています。

参加者が間違いを犯した場合は、その責任を負うことを期待します。誰かが傷ついたり不快な思いをしたりした場合は、注意深く敬意を持って耳を傾け、誤りを正すために最善を尽くすのが私たちの責任です。

このリストは網羅的なものではありませんが、私たちは年齢、文化、民族、性自認または表現、言語、出身国、政治的信念、職業、人種、宗教、性的指向、社会経済的地位、および技術的能力における多様性を明示的に尊重します。上記の保護されるべき特性に基づく差別（障害を持つ参加者への差別を含む）は容認しません。

問題の報告
----------------

不適切な行動を経験したり目撃したりした場合、またはその他の懸念事項がある場合は、[codeofconduct@squareup.com][codeofconduct_at] までメールで報告してください。詳細については、以下の報告ガイドラインをご覧ください。

謝辞
------

上記の声明やガイドラインにおけるアイデアや文言の一部は、[Twitter][twitter_coc]、[Ubuntu][ubuntu_coc]、[GDC][gdc_coc]、および [Django][django_coc] コミュニティの活動に基づいています。彼らの活動に感謝します。

報告ガイドライン
---------------

不適切な行動を経験したり目撃したりした場合、またはその他の懸念事項がある場合は、[codeofconduct@squareup.com][codeofconduct_at] までメールで報告してください。すべての報告は慎重に取り扱われます。

報告には以下の内容を含めてください。

 * あなたの連絡先情報。
 * 関与した個人の名前（本名、ニックネーム、またはペンネーム）。他にも目撃者がいる場合は、その情報も記載してください。
 * 何が起こったかの説明、およびその出来事が継続中であると思われるかどうか。公的な記録（メーリングリストのアーカイブや公開IRCロガーなど）がある場合は、リンクを含めてください。
 * その他、役立つと思われる追加情報。

報告の提出後、Square行動規範委員会の代表者から個人的に連絡があります。その後、委員会は出来事を調査し、必要に応じて追加の質問を行い、どのように対応するかを決定します。

不適切な行動を止めるよう求められた場合は、直ちに従うことが期待されます。個人が不適切な行動をとった場合、Square行動規範委員会は、警告なしにすべてのSquareのスペースから永久的に追放することを含む、適切と判断したあらゆる措置を講じることがあります。
