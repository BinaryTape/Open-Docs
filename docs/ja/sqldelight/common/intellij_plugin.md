# IntelliJプラグイン

IntelliJプラグインは、`.sq`ファイル向けの言語レベルの機能を提供します。その機能は以下の通りです。

 * シンタックスハイライト
 * リファクタリング/使用箇所の検索
 * コード補完
 * 編集後に`Queries`ファイルを生成
 * 右クリックで有効なSQLiteとしてコピー
 * IDE内のコンパイラエラーからファイルへのクリックジャンプ
