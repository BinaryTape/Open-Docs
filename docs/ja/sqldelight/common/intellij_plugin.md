# IntelliJ プラグイン

IntelliJ プラグインは、`.sq` ファイルに対して以下の言語レベルの機能を提供します：

 * 構文ハイライト
 * リファクタリング / 使用箇所の検索
 * コードの自動補完
 * 編集後の `Queries` ファイル生成
 * 右クリックによる有効な SQLite としてのコピー
 * IDE 内のコンパイルエラーからファイルへのジャンプ
