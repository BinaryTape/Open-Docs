# 与 Coil 配合使用

一组能与 Coil 良好配合的第三方库。

- [Telephoto](https://github.com/saket/telephoto)：在 Compose UI 中构建媒体体验的基础组件。
- [Landscapist](https://github.com/skydoves/landscapist)：一个可插拔且高度优化的 Jetpack Compose 和 Kotlin Multiplatform 图片加载库，支持通过 Glide、Coil 和 Fresco 获取并显示网络图片。

有与 Coil 配合使用或在其基础上构建的库吗？欢迎提交拉取请求 (PR) 将其添加到此处！
