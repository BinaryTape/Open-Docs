[//]: # (title: 类型概览)

在 Kotlin 中，一切皆为对象，这意味着你可以在任何变量上调用成员函数和属性。
尽管某些类型在运行时具有优化的内部表示形式（例如数字、字符和布尔值），它们作为基本类型值存在，
但对你而言，它们表现和行为都如同常规类。

本节描述了 Kotlin 中使用的基本类型：

*   [数字类型](numbers.md) 及其[无符号对应类型](unsigned-integer-types.md)
*   [布尔类型](booleans.md)
*   [字符类型](characters.md)
*   [字符串类型](strings.md)
*   [数组](arrays.md)

要了解其他 Kotlin 类型，例如 `Nothing`、`Any` 和 `Unit`，请查阅 Kotlin API 参考：

*   [`Any`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-any/)
*   [`Nothing`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-nothing.html)
*   [`Unit`](https://kotlinlang.org/api/latest/jvm/stdlib/kotlin/-unit/)

> [了解如何在 Kotlin 中执行类型检测和类型转换](typecasts.md)。
