您可以使用以下资料和资源来学习 Kotlin：
* [基础语法](basic-syntax.md) – 快速了解 Kotlin 语法。
* [Kotlin 之旅](kotlin-tour-welcome.md) – 学习 Kotlin 编程语言的基础知识。
* [惯用法](idioms.md) – 了解如何在常用场景中编写地道的 Kotlin 代码。
  * [Java 到 Kotlin 迁移指南：字符串](java-to-kotlin-idioms-strings.md) – 了解如何在 Java 和 Kotlin 中执行字符串相关的典型任务。
  * [Java 到 Kotlin 迁移指南：集合](java-to-kotlin-collections-guide.md) — 了解如何在 Java 和 Kotlin 中执行集合相关的典型任务。
  * [Java 到 Kotlin 迁移指南：为 null 性](java-to-kotlin-nullability-guide.md) — 了解如何在 Java 和 Kotlin 中处理为 null 性。
* [Kotlin Koans](koans.md) – 通过完成练习来学习 Kotlin 语法。每个练习都是一个失败的单元测试，您的任务是使其通过。推荐给具有 Java 经验的开发者。
* JetBrains Academy 的 [Kotlin Core 路线](https://hyperskill.org/tracks?category=4&utm_source=jbkotlin_hs&utm_medium=referral&utm_campaign=kotlinlang-docs&utm_content=button_1&utm_term=22.03.23) – 在逐步创建实际应用的过程中学习所有 Kotlin 核心要点。
* [Kotlin 书籍](books.md) – 寻找我们审阅过并推荐用于学习 Kotlin 的书籍。
* [Kotlin 技巧](kotlin-tips.md) – 观看短视频，Kotlin 团队将向您展示如何以更高效、更地道的方式使用 Kotlin，让您在编写代码时享受更多乐趣。
* [Advent of Code 谜题](advent-of-code.md) – 通过完成简短有趣的挑战，学习地道的 Kotlin 并测试您的语言技能。
* [Kotlin 实践教程](kotlin-hands-on.md) – 通过完成详尽的教程来全面掌握一项技术。这些教程将引导您完成与特定主题相关的独立项目。
* [面向 Java 开发者的 Kotlin](https://www.coursera.org/learn/kotlin-for-java-developers) – 在 Coursera 的这门课程中了解 Java 和 Kotlin 之间的异同。
* [PDF 格式的 Kotlin 文档](kotlin-pdf.md) – 离线阅读我们的文档。
