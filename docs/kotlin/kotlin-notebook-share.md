[//]: # (title: 分享您的 Kotlin Notebook)

要分享 [Kotlin Notebook](kotlin-notebook-overview.md)，您可以将其上传到任何 Notebook Web 查看器，因为 [Kotlin Notebook](data-analysis-overview.md#notebooks) 遵循通用的 Jupyter 格式。

我们推荐使用以下平台分享 Kotlin Notebook：

* **JetBrains Datalore：** 该平台不仅便于分享 Kotlin Notebook，还能提高其可用性。
  [Datalore](https://datalore.jetbrains.com/) 允许您执行和编辑 Notebook，并整合了高级功能，例如创建交互式报告和调度 Notebook 运行。欲了解其实际运行情况，请参阅 [使用 DataFrame 的 Kotlin Datalore 示例](https://datalore.jetbrains.com/report/static/KQKedA4jDrKu63O53gEN0z/B5YeMMONSAR78FgKQ9yJyW)。
  ![Datalore Notebook 示例](datalore-example.png){width=700}
* **GitHub**：GitHub 原生渲染 Kotlin Notebook，可以实现直接的分享与协作。
  示例请参阅 [Kotlin DataFrame GitHub 仓库示例](https://github.com/Kotlin/dataframe/blob/master/examples/notebooks/titanic/Titanic.ipynb)。
  ![GitHub Notebook 示例](github-notebook.png){width=700}
