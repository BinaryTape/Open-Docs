[//]: # (title: 使用 JetBrains Academy 插件学习 Kotlin)

通过在 [Android Studio](https://developer.android.com/studio) 和 [IntelliJ IDEA](https://www.jetbrains.com/idea/) 中均可使用的 [JetBrains Academy 插件](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy)，您可以通过代码练习任务来学习 Kotlin。

请查阅 [学习者入门指南](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy/docs/learner-start-guide.html?section=Kotlin%20Koans)，它将引导您开始在 IntelliJ IDEA 中学习 Kotlin Koans 课程。直接在 IDE 内解决交互式编程挑战并获得即时反馈。 

如果您想使用 JetBrains Academy 插件进行教学，请阅读[使用 JetBrains Academy 插件教授 Kotlin](edu-tools-educator.md)。
