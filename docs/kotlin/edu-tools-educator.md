[//]: # (title: 使用 JetBrains Academy 插件教授 Kotlin)

借助在 [Android Studio](https://developer.android.com/studio) 和 [IntelliJ IDEA](https://www.jetbrains.com/idea/) 中均可使用的 [JetBrains Academy 插件](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy)，您可以通过代码练习任务教授 Kotlin。

请参阅 [教育者入门指南](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy/docs/educator-start-guide.html?section=Kotlin)，了解如何创建一个包含一系列编程任务和集成测试的简单 Kotlin 课程。

如果您想使用 JetBrains Academy 插件学习 Kotlin，请阅读[使用 JetBrains Academy 插件学习 Kotlin](edu-tools-learner.md)。
