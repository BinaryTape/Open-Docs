[//]: # (title: Kotlin Night 指南)

Kotlin Night 是一种包含 3-4 场关于 Kotlin 或相关技术演讲的聚会。

> 关于 Kotlin Night 品牌资产，请参阅 [Kotlin 品牌资产文档](kotlin-brand-assets.md#kotlin-night-brand-assets)。
> 
{style="note"}

## 活动指南

* 请使用我们提供的[品牌材料](kotlin-brand-assets.md#kotlin-night-brand-assets)。让所有活动和材料保持相同的风格将有助于保持 Kotlin Night 体验的一致性。
* Kotlin Night 应该是一项免费活动。可以收取极少量的费用以支付开销，但它应始终保持为非营利性质。
* 活动应公开宣布，并向所有人开放参加，不得有任何形式的歧视。
* 如果您在活动结束后在线发布演讲内容，这些内容必须免费且可供所有人访问，无需任何注册或登记程序。
* 录像是可选的但建议进行，并且也应提供给大众。如果您决定录制演讲，我们建议制定一个计划以确保质量良好。
* 演讲应主要围绕 Kotlin，不应侧重于营销或销售。
* 活动可以视情况提供食物和饮料。

## 活动要求

JetBrains 非常乐意支持您的 Kotlin Night 活动。因为我们希望所有活动都能提供同样高质量的体验，我们要求组织者确保满足一些基本要求，以便活动获得 JetBrains 的支持。作为组织者，您负责活动的以下方面：

1. 举办活动所需的地点和所有必需品，包括预订一个舒适的场地。请确保：
    * 所有参与者都知晓活动的准确日期、地点和开始时间，以及活动的日程安排和项目。
    * 如果您提供食物和饮料，请确保有足够的空间以及充足的供应。
    * 您与演讲者之间已有计划。这包括演讲的日程、主题、摘要以及演示所需的任何必要设备。
2. 内容和演讲者
    * 您可以随意邀请来自当地社区、邻国甚至全球各地的演讲者。您的活动中不一定非要有 JetBrains 的代表或演讲者。然而，我们总是很高兴听到更多关于 Kotlin Night 的消息，所以请随时通知我们。
3. 公告与宣传
    * 至少在聚会日期的三周前宣布您的活动。
    * 在公告中包含日程、主题、摘要和演讲者简介。
    * 在社交媒体上进行传播。
4. 活动结束后向 JetBrains 提供活动材料
    * 我们很乐意在 [kotlinlang.org](https://kotlinlang.org/community/talks.html) 上宣布您的活动，如果您能为后续发布提供幻灯片和视频材料，我们将不胜感激。

## JetBrains 支持

JetBrains 提供以下支持：

* 获得 Kotlin Night 品牌的使用权，包括名称和徽标
* 周边，例如为演讲者准备的贴纸和 T 恤，以及为参与者准备的小纪念品
* 在 Kotlin Talks 页面上列出该活动
* 如有必要，协助联系演讲者参加活动
* 如果可能，协助寻找地点（通过联系人等），并协助确定与当地企业的潜在合作伙伴关系
