[Kotlin Notebook](kotlin-notebook-overview.md) 是一款交互式工具，允许你在一个文档中混合代码、可视化内容和 Markdown。你可以使用 Notebook 来编写和执行被称为 _代码单元格_ 的代码段，即时查看结果，并记录你的想法。这种设置使其成为快速原型开发、数据分析和数据科学的绝佳工具。
