[Kotlin Notebook](kotlin-notebook-overview.md) 是一款交互式工具，能让你在同一个文档中混合代码、视觉元素和 Markdown。你可以使用 Notebook 在被称为 _代码单元格_ 的部分编写和执行代码，立即查看结果，并记录你的想法。这种设置使其成为快速原型开发、分析和数据科学的优秀工具。
