# IntelliJ 플러그인

IntelliJ 플러그인은 `.sq` 파일에 대해 다음과 같은 언어 관련 기능을 제공합니다:

 * 구문 강조
 * 리팩터링/사용처 찾기
 * 코드 자동 완성
 * 편집 후 `Queries` 파일 생성
 * 마우스 우클릭으로 유효한 SQLite로 복사
 * IDE에서 컴파일러 오류 발생 시 클릭하면 파일로 바로 이동
