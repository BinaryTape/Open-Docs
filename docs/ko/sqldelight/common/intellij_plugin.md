# IntelliJ 플러그인

IntelliJ 플러그인은 `.sq` 파일에 대해 다음과 같은 언어 차원의 기능을 제공합니다:

 * 구문 강조(Syntax highlighting)
 * 리팩터링(Refactoring)/사용처 찾기(Find usages)
 * 코드 자동 완성
 * 편집 후 `Queries` 파일 생성
 * 우클릭하여 유효한 SQLite 문으로 복사
 * IDE 내 컴파일러 오류 클릭 시 해당 파일로 이동
