## Kotlin 로고

저희 로고는 마크와 서체로 구성됩니다. 
풀컬러 버전이 기본이며, 대부분의 경우에 이 버전을 사용해야 합니다.

[모든 버전 다운로드](https://resources.jetbrains.com/storage/products/kotlin/docs/kotlin_logos.zip){:.typo-float-right.kto-button.kto-button_size_m.kto-button_mode_outline}

로고와 마크에는 보호 영역(protective field)이 있습니다. 다른 디자인 요소가 이 영역을 침범하지 않도록 로고를 배치해 주세요.
보호 영역의 최소 크기는 마크 높이의 절반입니다.

![Kotlin 로고 비율](kotlin-logo-guidelines.png){width=700}

로고 사용과 관련하여 다음 제한 사항에 특히 주의해 주세요:

* 마크를 텍스트와 분리하지 마세요. 요소의 위치를 바꾸지 마세요.
* 로고의 투명도를 변경하지 마세요.
* 로고에 테두리를 두르지 마세요.
* 로고를 임의의 색상으로 다시 칠하지 마세요.
* 텍스트를 변경하지 마세요.
* 복잡한 배경 위에 로고를 배치하지 마세요. 밝은 배경 앞에 로고를 배치하지 마세요.

[Kotlin 브랜드 사용 가이드라인 읽어보기](https://kotlinfoundation.org/guidelines/).

## Kotlin 사용자 그룹 브랜드 자산

Kotlin 사용자 그룹이 쉽게 인식되고 Kotlin과의 연관성을 전달할 수 있도록 특별히 설계된 로고를 제공합니다.

* 공식 Kotlin 로고는 언어 자체와 연관되어 있습니다. 혼란을 줄 수 있으므로 
   다른 용도로 사용해서는 안 됩니다. 이는 로고를 변형한 형태에도 동일하게 적용됩니다.
* 사용자 그룹 로고는 커뮤니티의 의견과 활동이 Kotlin 팀과 독립적임을 의미하기도 합니다.
* 여러분의 의견이 저희와 일치할 필요는 없으며, 이것이 창의적이고 강력한 커뮤니티를 위한 가장 유익한 모델이라고 생각합니다.

[모든 자산 다운로드](https://drive.google.com/drive/folders/0B3Zi34svOj1RZ2sxZExhblRJc1k){:.typo-float-right.kto-button.kto-button_size_m.kto-button_mode_outline}

### 사용자 그룹 스타일

2017년 초 Kotlin 커뮤니티 지원 프로그램이 시작된 이후 사용자 그룹의 수가 급증하여 
매달 약 2~4개의 새로운 사용자 그룹이 합류하고 있습니다. 여러분의 지역에 있는 그룹을 찾으려면 
**Kotlin User Groups** 섹션의 전체 그룹 목록을 확인해 보세요.

새로운 Kotlin 사용자 그룹에 사용자 그룹 로고와 프로필 사진을 제공합니다.

![브랜딩 이미지](kotlin-user-group-logo.png){width=700}

이러한 지원을 하는 두 가지 주요 이유는 다음과 같습니다:

* 첫째, 공식 사용자 그룹으로 인식되는 데 도움이 되는 특별한 Kotlin 스타일 브랜드 자료를 
   요청하는 커뮤니티의 문의가 많았습니다.
* 둘째, 어떤 Kotlin 관련 자료가 공식 팀의 것인지, 어떤 것이 커뮤니티에서 제작된 것인지 명확히 구분할 수 있도록 
   사용자 그룹 및 커뮤니티 콘텐츠에 별도의 스타일을 제공하고자 했습니다.

### 사용자 그룹 로고 만들기

사용자 그룹 로고를 만들려면:
1. Kotlin 사용자 그룹 [로고 파일](https://docs.google.com/drawings/d/1IcJp8Z2jAwEliXrHB-l9RNK_2LrqGTkNuPPtjrW1iIU/edit)을 본인의 Google 드라이브로 복사합니다(Google 계정에 로그인되어 있어야 합니다).
2. **Your City** 텍스트를 사용자 그룹의 이름으로 바꿉니다.
3. 사진을 다운로드하여 사용자 그룹 자료에 사용합니다.

![벨라루스 Kotlin 사용자 그룹 샘플](kotlin-user-group-avatar.png){width=300}

*벨라루스 Kotlin 사용자 그룹 프로필 사진 샘플*

벡터 그래픽과 소셜 네트워크용 커버 사진 샘플이 포함된 [그래픽 세트](https://drive.google.com/drive/folders/0B3Zi34svOj1RZ2sxZExhblRJc1k)를 다운로드할 수 있습니다.

### 다양한 플랫폼용 그룹 프로필 사진 만들기

그룹의 프로필 사진을 만들려면:
1. Kotlin 사용자 그룹 프로필 [사진 파일](https://docs.google.com/drawings/d/1buhwccmllb7wFS0OIAub0WC4DIuSHRiDpjEQhB4tkPs/edit)의 사본을 본인의 Google 드라이브에 만듭니다(Google 계정에 로그인되어 있어야 합니다).
2. 사용자 그룹 위치의 약칭을 추가합니다(기본 샘플에 따라 최대 4개의 대문자 사용).
3. 사진을 다운로드하여 Facebook, Twitter 또는 기타 플랫폼의 프로필에 사용합니다.

### meetup.com 커버 사진 만들기

meetup.com용 그룹 로고가 포함된 커버 사진을 만들려면:
1. [사진 파일](https://drive.google.com/file/d/1g_0Plf_do6vrXvy1R-Hx430vfV2CPVKN/view)의 사본을 본인의 Google 
   드라이브에 만듭니다(Google 계정에 로그인되어 있어야 합니다).
2. 사진의 오른쪽 상단에 있는 로고에 사용자 그룹 위치의 약칭을 추가합니다. 
   일반적인 패턴을 사용자 지정 사진으로 바꾸려면 배경 패턴 이미지를 클릭하고 
   'Replace Image'(이미지 교체)를 선택한 다음 'Upload from Computer'(컴퓨터에서 업로드) 또는 다른 소스를 선택합니다.
3. 사진을 다운로드하여 [meetup.com](https://meetup.com)의 프로필에 사용합니다.

![사용자 그룹 예시](kotlin-user-group.png){width=700}
