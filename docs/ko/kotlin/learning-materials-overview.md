[//]: # (title: 학습 자료 개요)

Kotlin을 학습하기 위해 다음 자료와 리소스를 활용할 수 있습니다:
* [기본 문법(Basic syntax)](basic-syntax.md) – Kotlin 문법에 대한 빠른 개요를 확인하세요.
* [Kotlin 투어(Kotlin tour)](kotlin-tour-welcome.md) – Kotlin 프로그래밍 언어의 기초를 배웁니다.
* [Idioms(관용구)](idioms.md) – 일반적인 사례에 대해 관용적인(idiomatic) Kotlin 코드를 작성하는 방법을 배웁니다.
  * [Java에서 Kotlin으로의 마이그레이션 가이드: 문자열(Strings)](java-to-kotlin-idioms-strings.md) – Java와 Kotlin에서 문자열 관련 일반적인 작업을 수행하는 방법을 배웁니다.
  * [Java에서 Kotlin으로의 마이그레이션 가이드: 컬렉션(Collections)](java-to-kotlin-collections-guide.md) — Java와 Kotlin에서 컬렉션 관련 일반적인 작업을 수행하는 방법을 배웁니다.
  * [Java에서 Kotlin으로의 마이그레이션 가이드: Null 허용 여부(Nullability)](java-to-kotlin-nullability-guide.md) — Java와 Kotlin에서 null 허용 여부를 처리하는 방법을 배웁니다.
* [Kotlin Koans](koans.md) – Kotlin 문법을 익히기 위해 연습 문제를 완료하세요. 각 연습 문제는 실패하는 단위 테스트(unit test)로 작성되어 있으며, 여러분의 역할은 이를 통과시키는 것입니다. Java 경험이 있는 개발자에게 권장됩니다.
* JetBrains Academy의 [Kotlin Core track](https://hyperskill.org/tracks?category=4&utm_source=jbkotlin_hs&utm_medium=referral&utm_campaign=kotlinlang-docs&utm_content=button_1&utm_term=22.03.23) – 실제로 작동하는 애플리케이션을 단계별로 만들면서 Kotlin의 모든 핵심 요소를 배웁니다.
* [Kotlin 도서](books.md) – Kotlin 학습을 위해 저희가 검토하고 추천하는 도서들을 확인해 보세요.
* [Kotlin 팁](kotlin-tips.md) – Kotlin 팀이 Kotlin을 더 효율적이고 관용적인 방식으로 사용하는 방법을 보여주는 짧은 영상을 시청하세요. 더욱 즐겁게 코드를 작성할 수 있습니다.
* [Advent of Code 퍼즐](advent-of-code.md) – 짧고 재미있는 과제를 해결하며 관용적인 Kotlin을 배우고 언어 실력을 테스트하세요.
* [Kotlin 핸즈온 튜토리얼](kotlin-hands-on.md) – 특정 기술을 완전히 파악하기 위해 긴 호흡의 튜토리얼을 완료하세요. 이 튜토리얼들은 특정 주제와 관련된 독립된 프로젝트를 안내합니다.
* [Java 개발자를 위한 Kotlin](https://www.coursera.org/learn/kotlin-for-java-developers) – Coursera의 이 강좌에서 Java와 Kotlin의 유사점과 차이점을 배웁니다.
* [PDF 형식의 Kotlin 문서](kotlin-pdf.md) – 오프라인에서 문서를 읽어보세요.
