[//]: # (title: JetBrains Academy 플러그인으로 Kotlin 배우기)

[Android Studio](https://developer.android.com/studio) 및 [IntelliJ IDEA](https://www.jetbrains.com/idea/)에서 모두 사용 가능한 [JetBrains Academy 플러그인](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy)을 사용하면 코드 실습 과제를 통해 Kotlin을 배울 수 있습니다.

IntelliJ IDEA 내에서 Kotlin Koans 코스를 시작하는 데 도움이 되는 [Learner Start Guide](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy/docs/learner-start-guide.html?section=Kotlin%20Koans)를 살펴보세요.
IDE 내에서 바로 대화형 코딩 챌린지를 해결하고 즉각적인 피드백을 받을 수 있습니다. 

교육용으로 JetBrains Academy 플러그인을 사용하려면 [JetBrains Academy 플러그인으로 Kotlin 가르치기](edu-tools-educator.md)를 참고하세요.
