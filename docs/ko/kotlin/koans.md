[//]: # (title: Kotlin Koans)

Kotlin Koans는 주로 Java 개발자가 Kotlin 구문에 익숙해질 수 있도록 설계된 일련의 연습 문제입니다.
각 연습 문제는 실패하는 단위 테스트(unit test)로 구성되어 있으며, 여러분의 역할은 이 테스트를 통과하게 만드는 것입니다.

> 과제 내용은 [GitHub 저장소](https://github.com/Kotlin/kotlin-koans-edu)에서 확인할 수 있습니다.
> 
{style="note"}

Kotlin Koans 과제는 다음 중 한 가지 방법으로 진행할 수 있습니다:

* [Koans 온라인](https://play.kotlinlang.org/koans)에서 직접 풀어볼 수 있습니다.
* [JetBrains Academy 플러그인을 설치](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy/docs/install-jetbrains-academy-plugin.html)하고 [Kotlin Koans 코스를 선택](https://plugins.jetbrains.com/plugin/10081-jetbrains-academy/docs/learner-start-guide.html?section=Kotlin%20Koans)하여 IntelliJ IDEA 또는 Android Studio 내부에서 과제를 수행할 수 있습니다.

어떤 방식으로 Koans를 해결하든, 각 과제에 대한 솔루션을 확인할 수 있습니다:
* 온라인 버전에서는 **Show answer**를 클릭하세요.
* JetBrains Academy 플러그인의 경우, 먼저 과제를 직접 완료해 본 뒤 답변이 틀렸다면 **Peek solution**을 선택하세요.

과제를 구현한 후에는 솔루션을 확인하여 여러분의 답변과 제안된 답변을 비교해 보기를 권장합니다.
치팅(cheat)은 하지 마세요!
