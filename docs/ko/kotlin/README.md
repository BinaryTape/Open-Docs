이 문서는 Markdown 형식으로 작성되었으며, JetBrains에서 사용되는 일부 도메인 특화 언어(DSL) 구문을 포함하고 있습니다.

문서 작성에 기여하고 싶다면, [Kotlin 문서 가이드라인](https://docs.google.com/document/d/1mUuxK4xwzs3jtDGoJ5_zwYLaSEl13g_SuhODdFuh2Dc/edit)을 자세히 읽어 주시기 바랍니다.
