[Kotlin Notebook](kotlin-notebook-overview.md)을 공유하려면, [Kotlin 노트북](data-analysis-overview.md#notebooks)은 범용 Jupyter 포맷을 따르므로 단순히 모든 노트북 웹 뷰어에 업로드하기만 하면 됩니다.

Kotlin 노트북을 공유할 수 있는 플랫폼으로 다음을 권장합니다:

* **JetBrains Datalore:** 이 플랫폼은 Kotlin 노트북 공유를 용이하게 할 뿐만 아니라 사용성도 개선합니다. [Datalore](https://datalore.jetbrains.com/)를 사용하면 노트북을 실행 및 편집할 수 있으며, 대화형 보고서 생성 및 노트북 실행 예약(scheduling)과 같은 고급 기능을 제공합니다. 실제 작동 모습을 확인하려면 [DataFrame을 사용한 Kotlin Datalore 예제](https://datalore.jetbrains.com/report/static/KQKedA4jDrKu63O53gEN0z/B5YeMMONSAR78FgKQ9yJyW)를 참고하세요.
  ![Datalore Notebook example](datalore-example.png){width=700}
* **GitHub**: GitHub은 Kotlin 노트북을 네이티브하게 렌더링하여 간편한 공유와 협업을 지원합니다. 예제는 [Kotlin DataFrame GitHub 리포지토리 예제](https://github.com/Kotlin/dataframe/blob/master/examples/notebooks/titanic/Titanic.ipynb)를 참고하세요.
  ![GitHub Notebook example](github-notebook.png){width=700}
