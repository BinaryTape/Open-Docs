# 예제

Koog 프레임워크는 다양한 유스케이스(use case)에 맞는 AI 에이전트 구현 방법을 이해하는 데 도움이 되는 예제들을 제공합니다.
예제는 **Kotlin**과 **Java**로 모두 제공됩니다.
이 예제들은 여러분의 애플리케이션에 적용할 수 있는 주요 기능과 패턴을 보여줍니다.

아래의 예제들을 살펴보고 링크를 클릭하여 GitHub의 소스 코드를 확인해 보세요.
